# 国語・算数 プロンプトジェネレーター

特別支援学校（知的障害）の教職員向け、ノーコード（Vibe Coding）支援システム。
児童生徒の実態に合わせた **国語／算数・数学の学習アプリ（単一HTML）** を、AI（Google Gemini）に一発生成させるための「最適化システムプロンプト」を自動生成します。

## つかいかた

1. 教科（国語／算数・数学）を選ぶ
2. 段階 → ステップ／領域 → 詳細な指導内容 を順にプルダウンで選ぶ（**ラーニングマップ準拠**）
3. 指導目標・児童の興味関心や仕様の要望を自由記述する
4. 「最適化プロンプトを生成する」を押す
5. 画面下の **公開ウィザード** に沿って進める（Gemini → Googleサイト → iPad）

> ⚠️ 児童の個人名（実名）は絶対に入力しないでください。「児童」「Aさん」等に言い換えてください。

## 公開ウィザード（アプリ内ガイド）

PDFマニュアル『Geminiで作成した学習用アプリをiPadで使用する手順』の全工程を、アプリ内のチェックリスト付きウィザードとして内蔵しています。

**迷わないためのナビゲーション**
- 画面上部に「① プロンプト生成 → ② Gemini → ③ Googleサイト公開 → ④ iPad」の**フローステッパー**を常時表示。完了は緑✓、現在地は青で点滅表示され、クリックでその工程へ移動できます
- ステッパー直下の「**👉 つぎにやること**」バナーが常に次の1手を提示（クリックで該当箇所へ移動）
- ウィザードは**いま取り組む工程だけが開く**アコーディオン式。工程内の次の項目には「👈 つぎはこれ」が付き、全項目にチェックが付くと自動で次の工程が開きます

- **STEP1 プロンプト生成**：生成すると自動でチェックが付く
- **STEP2 Gemini**：「プロンプトをコピーして Gemini をひらく」ボタンでコピー＋起動を1クリック化。Canvas・思考モード・コード取得の手順とコツを表示
- **STEP3 Googleサイト**：ページ全体の埋め込み→ホームページ設定→共有設定→公開→リンク取得までの7手順を案内
- **STEP4 QRコード**：公開URLを貼り付けると **アプリ内でQRコードを生成**（Chromeの右クリック操作は不要）。印刷・PNG保存に対応。iPadの「ホーム画面に追加」までを案内

進捗（チェック状態）と公開URLはブラウザの localStorage に保存され、次回開いたときも保持されます。別のアプリを作るときは「進捗をリセット」を押してください。QRコードは端末内（`qrcode.min.js`）で生成され、URLを外部サービスに送信しません。

## ファイル構成

```
repository-root/        （GitHub管理 / GitHub Pages 公開）
├── index.html          本プロンプトジェネレーター（PC用UI・左右2カラム＋公開ウィザード）
├── qrcode.min.js       QRコード生成ライブラリ（qrcode-generator v1.4.4 / MIT License）
├── README.md
└── data/
    └── learning_map.json   ラーニングマップ準拠の全ステップJSONデータ
```

- 外部サーバーAPI（GAS等）に一切依存しない**スタンドアロン**構成。`fetch()` で `data/learning_map.json` を非同期取得して4連動プルダウンを生成します。

## 公開・配置

- **本体（このアプリ）：** GitHubリポジトリ（Private推奨）＋ GitHub Pages に配置し、教職員用PCブラウザで利用。
- **生成される教材アプリ：** iPad（Safari）向け。Googleサイトの「埋め込みコード」にHTMLを貼り付けて公開。

## ローカルで開くときの注意

`index.html` をファイル（`file://`）として直接開くと、ブラウザのセキュリティ制約で `fetch()` がブロックされデータを読み込めません。
ローカル確認時は簡易サーバー経由で表示してください（例：このフォルダで `python -m http.server` を実行し `http://localhost:8000/` を開く）。**GitHub Pages 上では問題なく動作します。**

## 収録範囲

国語・算数/数学とも **小学部1段階〜中学部2段階** を収録しています。

| 教科 | 段階 | グループ（ステップ／領域） |
|---|---|---|
| 国語 | 小学部1〜3段階 | STEP×領域（聞く・話す／読む／書く） |
| 国語 | 中学部1・2段階 | 領域（A 聞くこと・話すこと／B 書くこと／C 読むこと） |
| 算数・数学 | 小学部1〜3段階 | STEP×領域（数量の基礎／数と計算／図形／測定／データの活用） |
| 算数・数学 | 中学部1・2段階 | 領域（数と計算／図形／測定 or 変化と関係／データの活用） |

## データの編集と出典

`data/learning_map.json` は

```
subjects → 教科 → 段階 → グループ → details（詳細指導内容の配列）
```

の構造です。学校・児童の実態に応じて項目を自由に追記・編集できます。

**出典・生成方法**（`build_map.ps1` で再生成可能）:
- **小学部段階**：配布Excel原本『ラーニングマップ（小学部段階：国語／算数）』をパースして生成。
- **中学部段階**：『ラーニングマップ.pdf』のテキストレイヤーを抽出して生成（OCR由来の明白な誤字のみ補正）。出典は静岡大学『ラーニングマップ』（平成29年告示 特別支援学校学習指導要領及び解説に基づく）。
